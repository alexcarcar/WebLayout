Packages and Member Access

  • Visible within same class: private, default, protected, public
  • Visible within same package by subclass: default, protected, public
  • Visible within same package by non-subclass: default, protected, public
  • Visible within different package by subclass: protected, public
  • Visible within different package by non-subclass: public
