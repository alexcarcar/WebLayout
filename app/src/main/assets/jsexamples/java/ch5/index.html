Chapter 5 Self Test (p 178)

  1. Show two ways to declare a one-dimensional array of 12 doubles.

    • double d[] = new double[12];
    • double[] d = new double[12];

  2. Show how to initialize a one-dimensional array of integers to the values 1 through 5.

    • int i[] = {1, 2, 3, 4, 5};

  3. Write a program that uses an array to find the average of 10 double values. Use any 10 values you like.

    • FindAverage.java

  4. Change the sort in Try This 5-1 so that it sorts an array of strings. Demonstrate that it works.

    • BubbleStrings.java

  5. What is the difference between the String methods of indexOf() and lastIndexOf()?

    • str.indexOf(str2)finds the first index of str2 in str
    • str.lastIndexOf(str2)finds the last index of str2 in str

  6. Since all strings are objects of type String, show how you can call the length() and charAt() methods on this string literal: "I like Java".

    • System.out.println("I like Java".length()); // 11
    • System.out.println("I like Java".charAt(2)); // l

  7. Expanding on the Encode cipher class, modify it so that it uses an eight-character stringas the key.

    • Encode.java

  8. Can bitwise operators be appled to the double type?

    • Bitwise operations cannot be used on boolean, float, or double, or class types.

  9. Show how this sequence can be rewritten using the ? operator.

    • if (x < 0) y = 10; else y = 20;
    • y = (x < 0) ? 10: 20;

  10. In the following fragment, is the & a bitwise or logical operator? Why?

    • boolean a, b; ... if (a & b) ...
    • a & b acts like a because both a and b must be true for the if statement to be true

  11. Is it an error to overrun the end of an array? Is it an error to index an array with a negative value?

    • Array boundries are strictly enforced in Java; it is a run-time error to overrun or underrun the end of an array.

  12. What is the unsigned right-shift operator?

    • If you don't want to preserve the sign bit when shifting right, you can use an unsigned right shift >>> that always brings in a 0 on the left.

  13. Rewrite the MinMax class shown earlier in this chapter so that it uses a for-each style for loop

    • MinMax.java

  14. Can the for loops that perform sorting in the Bubble class be converted into for-each loops? If not, why not?

    • No, because the elements in a for-each loop are immutable.

  15. Can a String control a switch statement?

    • Yes, starting in JDK 7, Strings can control switch statements.
