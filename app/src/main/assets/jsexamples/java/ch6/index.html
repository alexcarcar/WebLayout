  7. If all objects of a class need to share the same variable, how must you declare that variable?

    • Declare a variable as static to share the variable among all objects of a class.

  8. Why might you need to use a static block?

    • When a class requires initialization before it is ready to create objects.

  9. What is an inner class?

    • An inner class is a non-static class defined inside another class

  10. To make a member accessible by only other members of its class, what access modifier must be used?

    • When a member of a class is specified as private, that member can be accessed only be other members of its class.
